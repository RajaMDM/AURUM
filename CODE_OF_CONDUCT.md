# Code of Conduct

Treat everyone as a competent professional.

- Challenge ideas, not people
- Critique architecture, not effort
- Assume good intent in all questions
- Feedback at peer level, not tutorial level

Not welcome: personal attacks, seniority gatekeeping, noise with no signal.

Maintained by [@RajaMDM](https://github.com/RajaMDM). Violations: thread closure, then block.
